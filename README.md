# Description

Code de scan 3D, cam1 et proj1 servent d'exemple

# Requirements

* CMake (3.1+)
* OpenCV 

# Plateforms

* Linux
* Windows

# Build

	git clone ...
	cd scan3d
	mkdir build
	cd build
	cmake ../
	make
	ln -s ../data data
	./main -elasmi -triangulate -scan
